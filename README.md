# Dzap

Projet Vue.js 3 avec Vite.

## Prérequis

- [Node.js](https://nodejs.org/) (v18 ou plus récent)

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvre [http://localhost:5173](http://localhost:5173) dans le navigateur.

## Build

```bash
npm run build
```

Les fichiers de production sont générés dans `dist/`.

## Aperçu du build

```bash
npm run preview
```

## Structure

```
dzap/
├── public/          # Fichiers statiques
├── src/
│   ├── App.vue      # Composant racine
│   └── main.js      # Point d'entrée
├── index.html
├── package.json
└── vite.config.js
```
